A comprehensive management system for a travel agency, featuring a robust MySQL database backend and a Java Swing administrative interface.

Key Features:
Database: Relational schema with stored procedures and triggers to manage trips, reservations, vehicle fleets, and accommodations.
GUI: A user-friendly dashboard for performing CRUD operations, assigning vehicles, and tracking agency logistics.

Tech Stack: Java, Swing, MySQL, JDBC.
